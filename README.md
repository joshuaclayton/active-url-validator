# Active URL Validator

A small CLI tool to validate URLs

## Usage

```sh
cat file_with_one_url_per_line | active-url-validator
```

## Install

### Rust

```sh
git clone git@github.com:joshuaclayton/active-url-validator.git
cd active-url-validator
cargo install --path .
```
